# Canal Dark de Inteligência Artificial - Kit Completo

Este repositório contém todo o material necessário para criar e lançar um canal dark no YouTube focado em Inteligência Artificial.

## Estrutura do Projeto

```
canal-dark-ia/
├── README.md                          # Este arquivo
├── identidade-visual/
│   ├── logos/                         # 3 opções de logo (NeuralDark, Dossiê IA, Protocolo IA)
│   ├── banners/                       # 3 opções de banner para o YouTube
│   └── thumbnails-modelo/             # 2 modelos de thumbnail
├── planejamento/
│   └── planejamento_estrategico.md    # Documento completo de estratégia
└── roteiros/
    └── roteiro_video_01_guia_ia_iniciantes.md  # Roteiro do primeiro vídeo
```

## Opções de Nome

| Nome | Estilo | Slogan |
|------|--------|--------|
| **NeuralDark** | Cyberpunk / Futurista | "Decodificando o Futuro da IA" |
| **Dossiê IA** | Investigativo / Misterioso | "Investigando a Inteligência Artificial" |
| **Protocolo IA** | Hacker / Terminal | "Acesso Concedido ao Futuro" |

| Cor | Hex | Uso |
|-----|-----|-----|
| Preto Profundo | `#0A0A0A` | Fundo principal |
| Azul Neon | `#00D4FF` | Destaques e acentos |
| Roxo Elétrico | `#8B5CF6` | Elementos secundários |
| Verde Matrix | `#00FF41` | Alternativa (estilo hacker) |
| Branco | `#FFFFFF` | Texto principal |
| Amarelo Alerta | `#FFD700` | CTAs e destaques em thumbnails |

## Ferramentas de Produção Recomendadas

| Função | Ferramenta | Custo |
|--------|-----------|-------|
| Roteiro | ChatGPT / Claude | Gratuito (com limites) |
| Narração (Voz IA) | ElevenLabs | Gratuito (10 min/mês) |
| Imagens | Leonardo AI / DALL-E | Gratuito (com limites) |
| Vídeo (B-roll) | Pexels / Pixabay | Gratuito |
| Edição | CapCut | Gratuito |
| Música | Biblioteca do YouTube | Gratuito |

## Como Começar

1. Escolha um dos 3 nomes e sua identidade visual correspondente
2. Crie o canal no YouTube e configure logo + banner
3. Leia o planejamento estratégico completo em `planejamento/`
4. Use o roteiro modelo em `roteiros/` como base para o primeiro vídeo
5. Produza o vídeo seguindo as notas de produção do roteiro
6. Publique e siga a estratégia de SEO descrita no planejamento
