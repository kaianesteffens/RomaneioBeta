# Referencias visuais do redesign Fretio

Este diretorio documenta as referencias visuais aprovadas pelo Eduardo para redesenho do app desktop PySide6.

As imagens originais foram geradas fora do repositorio e devem ser anexadas manualmente nas issues correspondentes quando for necessario renderizacao direta no GitHub.

Regra principal para agentes:

- As imagens aprovadas sao referencia visual obrigatoria.
- Nao fazer redesign livre.
- Nao alterar regra de negocio.
- Nao alterar providers, contratos de cotacao, updater, installer, workflows ou server.
- Implementar uma tela por issue.
- Preservar o comportamento existente e alterar somente apresentacao/layout/componentes visuais.
